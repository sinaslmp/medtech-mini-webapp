# Full-Stack MedTech Mini Web-App

A mini web-app that simulates medical image phases:
- **Arterial**: increases contrast
- **Venous**: applies gaussian smoothing

## Live Demo
- Frontend: https://sinaslmp.github.io/medtech-mini-webapp/
- Backend: https://sinaslmp-medtech-phase-backend.hf.space

## How to use
1. Upload a JPG/PNG image
2. Select a phase (Arteriosa / Venosa)
3. Click **Elabora immagine**
4. View original and processed images side-by-side

## Tech
- Frontend: Vue + Vite (GitHub Pages)
- Backend: FastAPI + Pillow (Hugging Face Spaces)
